## Getting Started 

1. duplicate .env.example and rename to .env
2. run this script
```bash

#run the docker for mongodb (optional)
docker-compose build
docker-compose up

#install npm package
npm install

#run node server
npm run dev
```
3. import postman collection from root directory to your postman
4. Ready to try!
